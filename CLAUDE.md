# Инструкция для Claude: подними коуча с памятью как Telegram-бота

Когда человек открывает этот репозиторий с тобой, помоги ему за ~20 минут поднять
личного коуча отдельным ботом в Telegram, который ПОМНИТ. Веди по одному шагу,
спрашивай по одному, редактируй файлы сам. Человек чаще всего не программист:
простыми словами, без жаргона, на его языке.

---

## СНАЧАЛА скажи, что это

Это **отдельный бот** в Telegram: человек пишет ему, бот отвечает голосом коуча.
Главное отличие от простого промпта — **память**:
- помнит переписку даже после перезапуска и передеплоя;
- держит связь между разговорами: сам сохраняет важное (паттерны, инсайты, факты)
  и возвращает их в новых разговорах.

Честно проговори:
- Работает 24/7, потому что живёт на сервере (Railway), а не на компе.
- Память включается, когда подключена база (Postgres). Без базы бот тоже
  запустится, но будет помнить только текущий запуск.
- Нужен платный баланс Anthropic (~$5 на старт), бот думает на Claude.

---

## Что понадобится (назови заранее)

- Telegram (для @BotFather и чтобы писать боту).
- Аккаунт [console.anthropic.com](https://console.anthropic.com) и ~$5 на балансе.
- Аккаунт [railway.app](https://railway.app) (вход через GitHub).
- ~20 минут.

---

## Сценарий. Иди по шагам, жди ответа после каждого

### Шаг 1. Создай бота (3 мин)
В Telegram найди **@BotFather**, `/newbot`, задай имя и username (кончается на
`bot`). BotFather пришлёт **токен** вида `7123456:AAH...` — это
`TELEGRAM_BOT_TOKEN`. В чат вставлять не нужно, пусть сохранит.

### Шаг 2. Получи ключ Claude (5 мин)
[console.anthropic.com](https://console.anthropic.com) → регистрация → **Plans &
Billing** (пополнить ~$5) → **API Keys** → **Create Key**. Это `ANTHROPIC_API_KEY`,
показывается один раз.

### Шаг 3. Узнай свой Telegram id (1 мин)
В Telegram найди **@userinfobot**, `/start`. Число `Id: 123456789` пойдёт в
`ALLOWED_USER_IDS`. Без него бот ответит кому угодно и чужие будут тратить баланс.

### Шаг 4. Залей на Railway (5 мин)
1. railway.app → войти через GitHub.
2. **New Project → Deploy from GitHub repo** → выбрать свой форк этого репо.
3. **Variables** → добавить:
   - `TELEGRAM_BOT_TOKEN` (шаг 1)
   - `ANTHROPIC_API_KEY` (шаг 2)
   - `ALLOWED_USER_IDS` (шаг 3)
   - `COACH_PROMPT` (личность коуча целиком, если хочет свою; иначе берётся
     `personality.md`)
4. Railway соберёт по Dockerfile и запустит.

### Шаг 5. Подключи память — базу (5 мин) ⭐
Это то, ради чего всё. Самый простой путь — Postgres прямо в Railway:
1. В том же проекте на Railway: **New → Database → Add PostgreSQL**.
2. Railway сам создаст переменную `DATABASE_URL` и свяжет её с ботом (если не
   связалась автоматически — в сервисе бота добавь переменную-ссылку на
   `DATABASE_URL` из Postgres).
3. Бот при старте сам создаст нужные таблицы.

(Альтернатива — бесплатный Supabase: Project → Settings → Database → Connection
string (URI) → вставить в `DATABASE_URL` бота.)

В логах бота при старте должно появиться «память включена (Postgres)». Если видишь
«память выключена» — значит `DATABASE_URL` не подцепился, проверьте переменную.

### Шаг 6. Проверь
Человек открывает бота в Telegram, пишет `/start`, потом любое сообщение. Бот
отвечает голосом коуча. Проверка памяти: расскажи боту что-то о себе, потом
напиши ещё раз позже (или после передеплоя) — он должен помнить.

## Личность коуча
По умолчанию берётся из `personality.md` (готовый тёплый формат). Если человек
хочет свой голос — отредактируй файл, либо вставь личность приватно в `COACH_PROMPT`
(тогда личное не попадёт в публичный репозиторий). Бот берёт `COACH_PROMPT` в
первую очередь.

## Безопасность
- Токен, ключ и `DATABASE_URL` — только в Variables на Railway, не в коде и не в
  гите.
- `ALLOWED_USER_IDS` заполнить обязательно.
- В базе лежит личная переписка — доступ к проекту Railway держать закрытым.
- `.env` и сессии в `.gitignore`.

## Если что-то не работает
Не торопись: смотрите логи в Railway вместе. Частое: забыта переменная, не
пополнен баланс Anthropic, или `DATABASE_URL` не подцепился (тогда память в
лёгком режиме, но бот отвечает). Объясняй простыми словами, предлагай переделать
шаг.
