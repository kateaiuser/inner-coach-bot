# Коуч в твоём Claude (на подписке, без API)

Если у тебя есть подписка Claude (Pro или Max), коуч может жить прямо в приложении:
без сервера, без API, без оплаты токенов. Два режима, можно оба.

## 1. Полный коуч — как Проект (текст, максимум глубины)

1. Открой [claude.ai](https://claude.ai) → создай **Проект** (например «Мой коуч»).
2. В настройках проекта (custom instructions) вставь целиком [personality.md](personality.md)
   — это личность коуча со всеми приёмами.
3. Для глубины добавь в **знания проекта** файлы из [coach-knowledge/](coach-knowledge/):
   `body-practices.md` (телесные практики пошагово) и `session-structure.md`
   (форматы сессий). Тогда коуч ведёт практики дословно.
4. Готово: каждый чат в этом проекте — твой коуч, и проект помнит историю.

Это и есть «со скиллами»: глубина лежит в знаниях проекта, коуч её подтягивает.

## 2. Голосовой коуч — через профиль (чтобы говорить голосом)

Голосовой режим видит только инструкции **профиля**, не проекта. Поэтому:

1. Claude → Settings → Profile → поле личных инструкций.
2. Вставь сжатую версию из [voice-coach.md](voice-coach.md).
3. Включи голосовой режим в приложении и говори.

Компромисс честный: голос = сжатый коуч (лимит профиля), полная глубина — в
текстовом Проекте.

## Приватность

Всё живёт в твоём приватном Claude. Личное о себе можешь дописать в инструкции
проекта — оно никуда не публикуется.

## Граница

Это коуч, не терапевт и не врач. При кризисе он направляет к живому специалисту.
